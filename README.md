## How to Use Frontend

- npm install
- npm run dev (1x compile)
- npm run watch (compile otomatis setiap ada perubahan)
- edit di file /resources/js

note: jangan edit langsung tanpa npm run watch / dev (app.css, main.css, app.js, main.js) -> di folder public

## Before Commit
 - npm run prod
